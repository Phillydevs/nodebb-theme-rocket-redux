    Rocket Theme for NodeBB

    • A theme with a card-based design for NodeBB forums. Based off persona theme and rocket theme rewritten by julianlam. Mobile enabled.

    • The "redux" version is the original rocket theme re-written with the Persona theme as a base.

    Screenshots of original by julianlam

    Installation

    Install via the "ACP/Plugins" page, or by executing npm install nodebb-theme-rocket-redux
